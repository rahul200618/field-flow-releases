# 🌾 AgroERP Desktop & Mobile Releases

Welcome to the official release repository for **AgroERP** by OpenAlgon! 

This repository serves exclusively as the fast, public distribution channel for our official application binaries. The core source code for AgroERP is securely maintained in a private enterprise repository.

## 📥 Download the App
You can always download the latest, most secure version of AgroERP directly from our official website:
👉 **[erp.openalgon.com](https://erp.openalgon.com)**

Alternatively, you can download the application directly from the [Releases Tab](../../releases/latest) in this repository:
- **Windows Desktop:** `AGROERP_x64-setup.exe`
- **Android Mobile:** `AGROERP.apk`

## 🚀 About AgroERP
AgroERP is the ultimate agricultural management software, featuring:
- Seamless GST Billing and Invoice Generation
- Real-time Inventory and Farm Management
- Cross-platform synchronization (Windows & Android)
- Beautiful, highly-optimized user interfaces
